ELAN  
=====

**ELAN** est un plugin QGIS d'aide à la décision pour une gestion intégrée des eaux urbaines par des solutions fondées sur la nature.

   **Auteurs** :

   * Gabrielle Favreau, Pascal Molle, Nicolas Forquet `REVERSAAL (INRAE) <https://reversaal.lyon-grenoble.hub.inrae.fr/>`_

   * Sophie Aubier, Jacky Volpes `Oslandia <https://oslandia.com/>`_

   **Anciens contributeurs** : Larissa Marcos, Marie-Chloé Rosse, Raphaël Delhome
