Préparation des données question du centralisé / décentralisé
=============================================================

**Préalable :**
* Télécharger QGIS depuis le `site officiel <https://qgis.org/download/>`_ et l'installer en suivant le `guide d'installation <https://qgis.org/resources/installation-guide/>`_.
* Prendre en main QGIS à l'aide de tutoriels (https://tutoqgis.cnrs.fr/ par exemple)
* Identifier les données SIG disponibles dans votre contexte et les télécharger (MNT, routes et bâtiments) : plateforme gouvernementale de mise à disposition de données, plugin QGIS existant.
* Chiffrer/estimer la population à raccorder
* :ref:`Installer Elan <_extension>`

Le logigramme ici présenté vise à vous guider pour préparer vos données avant de pouvoir créer des scénarios sur votre zone d'étude (:ref:`logigramme suivant <_flowchart-scenario>`).

Il est interactif : vous pouvez zoomer et cliquer sur certains blocs pour être renvoyé aux rubriques correspondantes dans la documentation.
