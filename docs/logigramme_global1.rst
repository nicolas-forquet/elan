Flux global question du centralisé / décentralisé
=================================================

Le logigramme ici présenté vise à vous donner une vue d'ensemble du flux de traitement pour traiter la question du centralisé / décentralisé : principales étapes, modules impliqués et interaction entre eux.

Il est interactif : vous pouvez zoomer et cliquer sur certains blocs pour être renvoyé aux rubriques correspondantes dans la documentation.

.. attention::
    Utiliser le mode clair pour une meilleure lisibilité du logigramme.
